Welcome

High-Performance Computing (HPC) applications solve complex scientific and engineering problems. However, writing parallel code does not automatically guarantee performance.

This website introduces the fundamental concepts of profiling, measuring, and tuning HPC applications using MPI, OpenMP, and hybrid models.

Why Profiling Matters

  • Identify bottlenecks
  • Improve scalability
  • Increase processor efficiency
  • Reduce communication overhead

Performance is engineered through measurement and iteration.
